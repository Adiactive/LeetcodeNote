# 69. Sqrt\(x\)

## Analysis

Use x / sqr = sqr instead of sqr \* sqr = x to avoid integer overflow or using long long.

## Code

### C++ 

```cpp
class Solution {
public:
    int mySqrt(int x) {
        if(x <= 1) return x;

        int l = 0, r = x;
        while(l <= r){
            int mid = (l + r) / 2;
            int quo = x / mid;
            if (quo < mid) r = mid - 1;
            else if (quo > mid) l = mid + 1;
            else return mid;
        }

        return r;
    }
};
```
